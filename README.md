## 豐原博愛段 851・852 地號 投資分析備忘錄

針對台中市豐原區博愛段 851、852 地號（信義街29巷1號，第二種商業區）所製作的不動產投資分析網頁，內容涵蓋標的速覽、基地與分區資訊、建照與規劃、市場成交比較、財務試算，以及簽約前盡職調查（Due Diligence）等項目。本備忘錄之數字以公開行情所做之估算為主，僅供內部投資評估參考，非正式投資建議。

網頁製作：初樸建築師事務所
